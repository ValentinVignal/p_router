Novade Solutions is a Singaporean company that provides a cloud-based platform for construction and real estate companies to manage their quality, safety and maintenance processes. I joined the company in 2019 as a software engineer and was promoted to senior software engineer in 2020.

The project I worked on is a mobile (Android and iOS) and web application that allows construction and real estate companies to manage their work. The front end (web and mobile) are developed using Flutter. The backend runs using TypeScript and node.js using Nestjs and TypeORM to connect to a PostgreSQL database. The API mostly uses GraphQL .

Novade Lite

I joined the project in 2021 around 1 year after it started and work on it for 3 years. I worked on it as a senior full stack developer designing and implementing the technical solutions. I spent the most time on the front end of the web application which was the most complex part of the project.

Some achievements

  • Re-organized the front-end code base into a mono repo using melos. It reduced the average time of the build runner from 40+minutes to around 10 minutes. It increased the caching performances and reduced the build and deployment time on the pipeline was reduced by 50%.
  • Organized and architected the mobile and web routing system using go_router.
  • Revamped and standardized the state management solution using riverpod which increased the overall speed and performances.
  • Revamped and re-designed the entire mobile and web application to use Flutter's theme.
  • Implemented a code generation tool to manage and use text localizations in a safe way which prevented numerous of recurring bugs.
  • Implemented the main and most used and complex feature of the application: the form template builder. Similar to Google forms, it allow the user to define a form's structure and then create and fill several instances of it.
  • Set up the frontend CI pipelines to ensure code style and tests success.
  • Implemented the front end part of the in-app payment feature using Stripe.
  • Kick started the use of Flutter tests which helped to prevent numerous bugs.
